Expense Manager

I've created a robust Password Manager using Python, which combines essential features for securely storing and managing your passwords. This application employs a SQLite database for efficient and secure password storage. The standout feature is the seamless integration of Google Drive, allowing you to upload and back up your password data to the cloud for added security and accessibility. Some key libraries and technologies used in this project include: - Logging: for effective error tracking and debugging. - Tkinter: for building a user-friendly graphical interface. - SQLite: for maintaining a reliable database for password storage. - Google Drive API: for secure cloud backup and synchronization.
